📖 Permission Guide
Level 1 (Creator):
  • CREATE_REQUEST: Create new requests
  • VIEW_REQUEST: View request details
Levels 2+ (Approvers):
  • APPROVE_TASK: Approve or reject tasks
  • QUERY_TASK: Raise queries
  • EDIT_REQUEST: Edit requests
  • VIEW_REQUEST: View requests
Final Level (Admin):
  • All approver permissions PLUS:
  • DELETE_REQUEST 🔑: Delete requests
  • VIEW_ALL_REQUESTS 🔑: View all requests
  • MANAGE_LEVELS 🔑: Admin functions
